# MLBoss

MLBoss is a Next.js-based fantasy-baseball decision-support tool that integrates Yahoo OAuth and an LLM-driven agent system.

## Quick Start

```bash
npm install                                   # install dependencies
docker run -d -p 6379:6379 redis:alpine      # start Redis
npm run dev                                   # start the dev server
```
